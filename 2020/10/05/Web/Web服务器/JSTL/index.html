JSTL
  1. 概念：JavaServlet Pages Tag Library JSP标准标签库，是由Apache提供的开源的免费的标签库
  2. 作用：用于简化和替换JSP页面上的java代码
  3. 使用步骤
    1. 导入JSTL相关的jar包
    2. 引入标签库：taglib指令 <%taglib %>
    3. 使用标签
  4. 常用的JSTL标签
    1. if：if语句
    2. choose：switch语句
    3. foreach：for语句
三层架构

  1. 界面层：用户看到的界面，用户可以通过界面上的组件与服务器进行交互

  2. 业务逻辑层：出来业务逻辑的。

  3. 数据访问层：操作数据存储文件。
